# Prun Tools

A collection of tools for the game [Prosperous Universe](https://prosperousuniverse.com/)

## Development

- Initial setup:

  ```sh
  npm run setup
  ```

- Start dev server:

  ```sh
  npm run dev
  ```

This starts your app in development mode, rebuilding assets on file changes.
